## Install

```cs
#addin nuget:?package=Cake.AsciiDoctorJ
```

## Usage

```cs
#addin nuget:?package=Cake.AsciiDoctorJ

Task("MyTask").Does(() => {
  AsciiDoctorJ(s => s
	  .WithVerbose()
	  .WithInputFile(file)
	  .WithDestinationDir(distDir));
});
```
